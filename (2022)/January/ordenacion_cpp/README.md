# Sorting Algorithm in C++

This project implements a sorting algorithm in C++ to order an array. You can choose between **Bubble Sort** and **QuickSort**. This project helps to practice array manipulation and control structures in C++.

## Instructions

1. Compile `ordenacion.cpp`.
2. Run the program to see the sorted output of the array.

## Dependencies
- C++ Compiler (e.g., g++)
