# Summary Page with Section Navigation in JavaScript

This project is a summary page that allows users to navigate between multiple sections using JavaScript. It is ideal for displaying divided content in an easily accessible format.

## Instructions

1. Open the `index.html` file in a web browser.
2. Use the buttons to navigate between different sections.

## Features

- **Section Navigation:** Allows switching between sections of content.
- **JavaScript Control:** Uses JavaScript to manage section visibility.

## Dependencies

- Web browser (e.g., Chrome, Firefox)

# Página de Resumen con Navegación entre Secciones en JavaScript

Este proyecto es una página de resumen que permite a los usuarios navegar entre varias secciones utilizando JavaScript. Es ideal para mostrar contenido dividido en un formato fácilmente accesible.

## Instrucciones

1. Abra el archivo `index.html` en un navegador web.
2. Use los botones para navegar entre diferentes secciones.

## Funcionalidades

- **Navegación entre Secciones:** Permite cambiar entre secciones de contenido.
- **Control con JavaScript:** Utiliza JavaScript para gestionar la visibilidad de las secciones.

## Dependencias

- Navegador web (por ejemplo, Chrome, Firefox)
